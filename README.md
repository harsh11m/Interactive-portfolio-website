# Interactive Portfolio Website

## Description

This project is a responsive and interactive personal portfolio website developed using **HTML, CSS, and JavaScript**. It showcases my profile, skills, projects, and contact information while focusing on clean design, responsiveness, and user interaction.

## Features

* Responsive design for different screen sizes
* Interactive navigation menu
* Smooth scrolling between sections
* Hover effects and animations
* Skills, Projects, About, and Contact sections
* Clean and organized code structure

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

```
Interactive-Portfolio-Website/
│── index.html
│── style.css
│── script.js
│── assets/
│   ├── images/
│   └── icons/
└── README.md
```

## How to Run

1. Clone the repository.
2. Open the project folder.
3. Run `index.html` in any modern web browser.
